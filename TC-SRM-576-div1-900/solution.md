#CharacterBoard
很久很久以前，有一个活泼可爱的小朋友，他拥有一个$$1000000000$$行$$W$$列的棋盘，行从上至下以$$0..999999999$$编号，列从左至右以$$0..W-1$$编号。
后来，有一个长者给了他一个长度为$$L$$的字符串$$s[0..L-1]$$，并让小朋友以以下伪代码去填充这个棋盘$$a[1000000000][W]$$。
```C++
int cursor = 0;
for (int i = 0; i < 1000000000; i++)
    for (int j = 0; j < W; j++)
    {
        a[i][j] = s[cursor];
        cursor = (cursor + 1) % L;
    }
```
